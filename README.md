# First time setup

### Create virtual environment

```python
py -3.11 -m venv venv
```

Activate environment

```python
venv/Scripts/activate
```

Upgrade pip

```python
py -m pip install --upgrade pip
```

Install requirements

```python
py -m pip install -r requirements.txt
```
